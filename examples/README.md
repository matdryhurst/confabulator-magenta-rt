# Examples

This directory contains example applications and plug-ins that consume the `magentart::core` inference engine.

## Subdirectories

- [`mrt2/`](mrt2/) — Main plugin: standalone host, AUv3 host.
- [`jam/`](jam/) — App for exploring note control.
- [`collider/`](collider/) — App for exploring prompt space.
- [`hello_mrt2/`](hello_mrt2/) — Minimal CLI application that loads a model and exports generated audio as a WAV file. Useful as the shortest starting point to understand the core API.
- [`max/`](max/) — `mrt~` MaxMSP signal external (`.mxo`).
- [`pd/`](pd/) — `mrt~` Pure Data signal external (`.pd_darwin`).
- [`sc/`](sc/) — `MagentaRT` SuperCollider UGen plugin (`.scx`).
